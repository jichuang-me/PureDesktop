# PureDesktop 使用手册

欢迎使用 PureDesktop！本手册将帮助您快速上手并精通这款桌面整理工具。

## 1. 快速入门

### 安装

1. 下载最新版本的 `PureDesktop.zip`。
2. 解压到任意文件夹（建议不要放在桌面）。
3. 运行 `PureDesktop.exe`。

### 首次使用

启动后，您的桌面图标将保持原样。

- **一键整理**：右键点击右下角托盘图标 -> 选择“一键整理”。程序将自动按文件类型创建“格子”并分类。
- **显示/隐藏**：双击桌面空白处，或左键点击托盘图标。

## 2. 核心概念：格子 (Fences)

“格子”是 PureDesktop 的核心单位。

- **自动格子**：由“一键整理”生成，根据内部算法分类。
- **映射格子**：手动添加，同步显示特定文件夹中的内容。
  - *添加方式*：右键托盘图标 -> “添加映射格子” -> 选择文件夹。

### 分类逻辑

- **快捷方式**: .lnk, .url
- **文档**: .doc, .docx, .txt, .pdf, .xlsx, .pptx 等
- **图片**: .jpg, .png, .gif, .bmp, .ico 等
- **程序**: .exe, .msi
- **压缩包**: .zip, .rar, .7z

## 3. 进阶设置

### 排除规则 (Exclusions)

防止特定项目被移动：

1. 进入“设置” -> “排除”。
2. **后缀排除**：输入如 `.tmp, .log`。
3. **名称排除**：输入如 `工作简报.docx` 或某个文件夹名。

### 视觉自定义

- **透明度**：在格子右键菜单中调节单个格子透明度，或在托盘设置中调整全局。
- **亚克力效果**：默认启用，让桌面更具设计感。
- **折叠模式**：点击格子标题可快速收起，只留标题栏。

## 4. 常见问题 (FAQ)

**Q: 点击“一键整理”后图标不见了？**
A: 请检查是否误操作隐藏了格子（双击桌面试试），或检查格子是否重叠。

**Q: 软件会开机自启吗？**
A: 您可以在托盘菜单中勾选“开机自启”。

**Q: 如何删除一个格子？**
A: 右键点击格子标题栏 -> 选择“删除格子”。注意：普通格子删除不会删除其中的文件，但映射格子删除需谨慎操作。
