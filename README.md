## English Description

PureDesktop is a lightweight, modern Windows desktop organizer designed to keep your workspace clean and productive. It automatically classifies your files into themed "fences" and stays invisible until you need it.

### ğŸŒŸ Key Features

- **Smart Classification**: Automatically organizes desktop files into categories (Word, Pictures, Programs, etc.).
- **Folder Mapping**: Map any folder on your PC into a desktop fence with real-time synchronization.
- **Taskbar Silence**: Completely hidden from the taskbar and Alt+Tab list for a true "pure" desktop experience.
- **Smart Snapping**: Fences snap to screen edges and other fences for perfect alignment.
- **Auto-Hide**: Fences can automatically fade out when not in use to reveal your wallpaper.
- **Global Toggle**: Quickly show/hide all fences by left-clicking the tray icon or double-clicking the desktop.
- **Dark Mode Support**: Full Windows Dark/Light mode support with high-contrast UI.
- **Single Instance**: Prevents multiple processes from running simultaneously.

### âŒ¨ï¸ Keyboard Shortcuts

| Shortcut | Action |
| :--- | :--- |
| **F2** | Rename selected item |
| **Alt + Enter** | Open file properties dialog |
| **Delete** | Move selected item(s) to Recycle Bin |
| **Ctrl + C / X** | Copy / Cut selected item(s) |
| **Ctrl + V** | Paste files into a mapped fence (or to desktop) |

### ğŸ–±ï¸ Tray Operations

- **Left Click**: Toggle fence visibility.
- **Right Click**: Open menu for settings, adding mapped fences, theme switching, and language selection.

### ğŸ› ï¸ Build & Run

- Requires **.NET 8 SDK**.
- Run `dotnet build` in the `PureDesktop` directory.
- Launch `PureDesktop.exe` from the output folder.
